# Week 4 - Country Explorer

## Project Goal
Build a Country Explorer app that requests country data from an API and displays the result on the page.

This project focuses on API requests, `fetch`, `async` / `await`, JSON, error handling, data extraction, and DOM updates.

## Instructions
Work in the `starter` folder of this repository.

## Challenges

1. Store the searched country name.
2. Build the country API URL.
3. Request country data.
4. Check whether the request failed.
5. Convert the country response to JSON.
6. Choose the matching country.
7. Request a country image.
8. Store the image URL.
9. Store the country name.
10. Store the capital.
11. Store the region.
12. Store the currency.
13. Store the formatted population.
14. Display the country name.
15. Display the capital.
16. Display the region.
17. Display the population.
18. Display the currency.
19. Display the country image.

## Extension Challenges
After the required app works, you may personalize the existing project without changing the main behavior.
